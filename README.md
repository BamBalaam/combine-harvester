# Combine Harvester

CLI tool to log Harvest timesheets with FZF

## Installation

```
pip install combine-harvester
```

## Requirements

* Python 3.6 (f-strings!)
* [jq](https://github.com/stedolan/jq)
* [fzf](https://github.com/junegunn/fzf)

## Environment Variables

* HARVEST_ACCOUNT_ID
* HARVEST_ACCOUNT_TOKEN

## Usage

```
> harvest
```
